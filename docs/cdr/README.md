# Convention Decision Records

Ratified conventions — style, structure, tooling. The convention counterpart to [ADRs](../adr/) (which ratify *architectural* decisions). CDRs are promoted from a [TN](../tn/) once agreed. Format `CDR-NNNN`, zero-padded, no year reset.
